# Hướng Dẫn Triển Khai Dự Án

# Đối với source code FE:

## Cài Đặt
Để thiết lập dự án, hãy thực hiện các bước sau:

1. Clone repository:
   ```bash
   git clone https://gitlab.com/kltl-ute/242k/19-fe

2. Cài đặt các gói phụ thuộc:
   Sử dung: yarn install (hoặc npm install)

3. Chạy dự án:
   Sử dung: yarn dev (hoặc npm run dev)
